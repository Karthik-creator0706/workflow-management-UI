# Workflow Management System UI

## Live Demo
Frontend (Versel): https://workflow-management-ui-six.vercel.app/
Frontend-Deploy(Render) : https://workflow-management-ui.onrender.com

## Project Overview
A Role-Based Workflow Management System built using React, Spring Boot, MySQL, and JWT Authentication.

- View Requests

- Workflow Logs
- JWT Authentication
